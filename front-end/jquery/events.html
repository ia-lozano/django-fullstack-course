jQuery Events

You can use the $ sign to select elements from the DOM! it will feel a lot like selecting using querySelectorAll(), for example

  • Selecting all a tags:
    • $('a')
  • Selecting all elements from the class "container":
    • $('.container')
  • Selecting all elements with id "special"
    • $('#special')
